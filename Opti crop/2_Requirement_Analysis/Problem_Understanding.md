# Problem Understanding

## Problem Statement

Agriculture is one of the primary sources of livelihood worldwide. Farmers often select crops based on traditional knowledge rather than scientific analysis of soil nutrients and environmental conditions.

Incorrect crop selection leads to:

- Low crop yield
- Poor soil utilization
- Financial losses
- Inefficient farming practices

## Proposed Solution

The OptiCrop system uses Machine Learning to recommend the most suitable crop based on:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

The recommendation is generated using a trained Logistic Regression model and displayed through a Flask web application.

## Expected Benefits

- Improved crop productivity
- Scientific crop recommendations
- Better resource utilization
- User-friendly web application
