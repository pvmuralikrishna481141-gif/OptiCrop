# Dataset Description

## Dataset Information

The Crop Recommendation dataset contains agricultural parameters collected from different soil conditions.

| Feature | Description |
|---------|-------------|
| Nitrogen | Nitrogen level in soil |
| Phosphorus | Phosphorus level in soil |
| Potassium | Potassium level in soil |
| Temperature | Temperature (°C) |
| Humidity | Relative Humidity (%) |
| pH | Soil pH |
| Rainfall | Rainfall (mm) |
| Label | Recommended Crop |

## Dataset Characteristics

- Clean dataset
- No missing values
- Numerical features
- One categorical target variable
- Suitable for classification problems
