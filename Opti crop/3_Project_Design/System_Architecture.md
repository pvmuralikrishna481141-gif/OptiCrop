# System Architecture

## Workflow

The application follows the architecture below.

1. User enters soil nutrient values.
2. Flask receives the input.
3. Input values are validated.
4. Logistic Regression model predicts the crop.
5. Predicted crop is displayed on the webpage.

## Components

- HTML User Interface
- Flask Backend
- Logistic Regression Model
- Crop Recommendation Dataset
- Prediction Result
