# Risk Analysis

## Possible Risks

### Data Quality Issues

Poor quality data may reduce prediction accuracy.

Mitigation:
Perform preprocessing and data cleaning.

---

### Model Accuracy

The selected algorithm may not achieve the desired accuracy.

Mitigation:
Compare multiple machine learning algorithms before selecting the best model.

---

### Deployment Errors

The web application may fail during deployment.

Mitigation:
Use Render cloud platform and verify dependency installation.

---

### User Input Errors

Users may enter invalid values.

Mitigation:
Implement backend validation in Flask.
