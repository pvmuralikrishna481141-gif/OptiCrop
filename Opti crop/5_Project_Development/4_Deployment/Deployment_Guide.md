# Deployment

## Platform

Render Cloud Platform

## Repository

GitHub

## Deployment Steps

1. Upload project to GitHub
2. Connect GitHub repository to Render
3. Configure Build Command

pip install -r requirements.txt

4. Configure Start Command

gunicorn app:app

5. Deploy application
